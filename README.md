# Portfolio Website

Static portfolio website built with HTML, CSS, and JavaScript.

## Pages
- index.html
- projects.html
- project.html
- admin.html

## Data Flow
- Default content is loaded from /data JSON files.
- Any admin edits are saved to localStorage.
- You can export/import the data from the admin page.

## Admin Password
Default: admin123

## Run Locally
Use a simple local server so JSON fetch works correctly.
